| Phase   | Days   | Focus                                  |
| ------- | ------ | -------------------------------------- |

### 🎯 Goal:

Build strong base in coding + mathematical intuition required for ML

---

### 📅 Plan

**Day 1–3**

* Python basics (variables, loops, conditionals)
* Solve basic problems

**Day 4–6**

* Functions, data structures, OOP basics
* Exception handling

**Day 7–8**

* NumPy (arrays, operations)

**Day 9–10**

* Pandas (dataframes, cleaning)

**Day 11–13**

* Statistics:

  * mean, variance, distributions
  * probability basics
  * Bayes theorem

**Day 14–17**

* Linear Algebra:

  * vectors, matrices
  * dot product
  * matrix operations

**Day 18–19**

* Calculus:

  * derivatives
  * partial derivatives
  * gradient intuition

**Day 20 (⚠️ Reinforcement)**

* Mini pipeline:

  * load dataset → clean → analyze
