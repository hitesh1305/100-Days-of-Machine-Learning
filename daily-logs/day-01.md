# Day 01 — Python Basics & Problem Solving

---

## 📌 Objective

Build foundational Python knowledge and start developing problem-solving skills.

---

## 🧠 What I Learned

### 🔹 Core Concepts

* Variables and data types (int, float, string)
* Taking input and printing output
* Conditional statements (`if`, `elif`, `else`)
* Loops:

  * `for` loop (fixed iterations)
  * `while` loop (condition-based iterations)

---

### 🔹 Key Understanding

* Programming is about **breaking a problem into small logical steps**
* Loops help automate repetitive tasks
* Conditions control the flow of logic

---

## ⚙️ Problems Solved

### 1. Even or Odd Number

* Used modulo operator (`%`)
* Logic: `n % 2 == 0`

---

### 2. Largest of Three Numbers

* Used nested `if-else`
* Compared values using logical `and`

---

### 3. Sum of First N Numbers

* Used loop to accumulate values
* Introduced concept of a running total

---

### 4. Factorial of a Number

* Used loop multiplication
* Understood iterative vs mathematical thinking

---

### 5. Reverse a Number

* Used modulo (`%`) and integer division (`//`)
* Extracted digits step-by-step

---

### 6. Palindrome Number

* Compared original number with reversed number
* Combined multiple concepts (loops + logic)

---

## 🔍 Key Insights

* Writing logic is harder than writing syntax
* Once logic is clear, code becomes easy
* Breaking problems into smaller steps simplifies everything

---



## 🔁 How I Solved Them

* Dry-run problems manually
* Printed intermediate values to debug
* Rewrote logic in simpler steps
